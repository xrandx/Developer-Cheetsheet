# 基础

## 移动

### 前缀可加数字 N

#### 按字符移动

`h`向左移动

`j` 向下移动

`k` 向上移动

`l` 向右移动

#### 按单词移动

`b` 向前移动到第 N 个单词的开头 (begin)

`B` 向前移动到第 N 个单词的开头，忽略分割符

`w` 向后移动到第 N 个单词的开头 (word)

`W` 向后移动到第 N 个单词的开头 ，忽略分割符

`e` 向后移动到第 N 个单词的结尾 (end)

`E`向后移动到第 N 个单词的结尾 (end)，忽略分割符

#### 移动到极端位置

`0` 移动到当前行的第一个字符处 (Home key)

`^`移动到当前行的第一个非空字符处
